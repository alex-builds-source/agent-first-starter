# agent-first-starter v0.1.4

## What shipped
- Scaffold now includes `CONTRIBUTING.md` by default with contribution/release hygiene guidance.
- Added `LICENSE` to strengthen publish-readiness defaults in the starter baseline.
- Added test coverage to verify CONTRIBUTING quality-check content.
- README scaffold template now points contributors to `CONTRIBUTING.md`.

## Why it matters
- New repos start with clearer maintainer guardrails from day zero.
- The scaffold reduces first-release friction by including contribution, license, security, and test hygiene defaults.

## Validation
- `PYTHONPATH=src pytest -q` → `5 passed`
- `PYTHONPATH=src python -m repo_preflight.cli check --path . --strict --compact` → `pass=18 warn=0 fail=0`
- `gitleaks git --redact` → no leaks found
